1. 개인정보의 처리 목적

  • 회원 식별 및 인증
  • 프로젝트 게시 및 투표 집계
  • 서비스 개선

2. 처리하는 개인정보 항목

필수 이메일 주소
필수 OAuth 식별자 (GitHub/Google)
필수 닉네임(displayName)
선택 프로필 이미지
필수 소속 클래스
선택 제출 프로젝트(제목/URL/태그라인/스크린샷)
자동 투표 기록
자동 접속 로그 (IP, User-Agent)

3. 개인정보의 처리 및 보유 기간

회원 탈퇴 시 지체 없이 파기합니다. 관계 법령에 따른 보존 의무가 있는 경우에 한해 해당 기간 동안 보관 후 파기합니다.

4. 개인정보의 제3자 제공

본 서비스는 회원의 개인정보를 제3자에게 제공하지 않습니다.

5. 개인정보 처리의 위탁

수탁자 위탁 업무
Supabase 인증 · 데이터베이스 · 스토리지
Vercel 호스팅 · 배포
Google OAuth 인증 제공
GitHub OAuth 인증 제공

6. 정보주체의 권리와 행사 방법

  • 열람 · 정정 — 설정 페이지
  • 삭제(탈퇴) — 설정 페이지의 ‘회원 탈퇴’
  • 처리 정지 요구 — 이메일 문의

7. 개인정보의 파기 절차 및 방법

데이터베이스 레코드는 전자적으로 삭제하며, 스토리지 파일은 제거합니다. 탈퇴 시 즉시 실행됩니다.

8. 개인정보의 안전성 확보 조치

  • 전 구간 HTTPS
  • Row-Level Security (Supabase)
  • 암호화된 스토리지
  • 운영자 키의 최소 권한 원칙

9. 자동 수집 (쿠키 등)

필수 Supabase 인증 세션 쿠키 — 로그인 유지
통계 Vercel Analytics — 익명 이용 통계
통계 Vercel Speed Insights — 성능 지표

개인 식별 정보는 수집하지 않습니다.
