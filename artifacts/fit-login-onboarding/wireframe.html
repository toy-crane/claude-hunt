Notes · Scenarios 3 & 4 · shared shell + header rhythm

Visual verification: both screens must share identical outer shell and header rhythm.

  • Shell: same bg, same max-w-sm form width, same vertical/horizontal padding
  • Header rhythm: Logo (text-3xl) → h1 (text-2xl, mt-8) → subtitle (text-sm muted, mt-1) → form (mt-8)
  • Motion: underscore blinks at the same 1s cadence on both
  • Divergence allowed: only inside the form body (controls, copy, buttons)
