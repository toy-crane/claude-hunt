# Korean Translation — Clarification Notes

## Core Principle

"수강생의 일상 어휘"가 기준. 모든 영어를 기계적으로 한국어로 바꾸지 않고, **한국 개발자가 평소 대화에서 그 단어를 영어로 말하는지**가 판단 기준.

## Scope

| 분류 | 처리 | 예시 |
|------|------|------|
| UI 문구 | 전부 한국어화 | 안내 메시지, 버튼 텍스트, 일반 메뉴 문구 |
| 사용자 콘텐츠 | 손대지 않음 (이미 한국어) | 프로젝트 제목/설명, 프로필 bio |
| 개발자 일상 영어 | 영어 유지 | `Upvote`, `Project`, `Profile` |
| 브랜드/고유명사 | 영어 유지 | `Claude Hunt`, `Claude Code`, `Built by toycrane` |
| 비(非)개발자 영어 어휘 | 한국어화 | `Cohort` → **클래스** |

## Key Decision: Cohort → "클래스"

검토한 대안: 시즌 / 반 / 배치 / 코호트 / 클래스 / 챕터 / 라운드 / 웨이브

"클래스"가 "딱딱하다"고 느껴졌지만, 이 서비스 맥락에서는 그 무게가 오히려 **"우리 3기"라는 소속감과 정체성**으로 기능한다는 점에서 적합하다고 판단.

## Out of Scope

- 언어 전환(i18n) 기능 — 수강생이 전원 한국인이므로 불필요
- 사용자 생성 콘텐츠 번역 — 이미 한국어로 작성됨
