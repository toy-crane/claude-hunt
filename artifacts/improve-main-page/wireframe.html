Notes

뷰포트 폭이 720px 미만일 때의 스택 레이아웃 (🅰️안 — 보수적 정리)

  • 진입: 좁은 뷰포트에서 동일 URL 접속 (또는 리사이즈)
  • 테이블 헤더 (RANK/PREVIEW/NAME/AUTHOR/VOTES) 미표시
  • 칩: flex-wrap으로 줄바꿈 — 가로 스크롤바 없음
  • 카드 행: 높이 64px 고정 · 썸네일 48×48 정방형 | 텍스트 블록(3줄) | 가로형 vote 버튼
  • rank 배지가 썸네일 좌상단 모서리에 오버레이 — 별도 rank 컬럼 제거
  • 텍스트 3줄: 제목 · 태그라인(dim) · 작성자 · 시간(dim) — 모두 1줄 truncate
  • vote 버튼은 가로형(↑ 12) · outline/filled 두 상태 (Row 02에 voted 상태 시각화)
  • 프롬프트 라인은 길어지면 줄바꿈 · 터라코타 $ 유지
  • "프로젝트 제출" 버튼은 제목 아래 왼쪽 정렬
  • Footer는 세로 스택
