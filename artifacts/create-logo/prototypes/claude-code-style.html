Claude Hunt — Claude Code style

Terminal-native: monospace wordmark, terracotta asterisk (✳), ASCII-art flavor. Four variants to pick from.

Variant A

✳ claude-hunt

The simplest Claude Code signature: a terracotta asterisk followed by a lowercase kebab-case monospace wordmark. Reads like a CLI tool.

~
✳ claude-hunt
Discover what the cohort is building.
