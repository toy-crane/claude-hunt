# Glossary

claude-hunt는 Claude Code 수업 수강생들이 만든 프로젝트를 모아 보여주고 서로 추천하는 보드다.

**클래스**:
수강생이 소속되는 한 무리. 사용자에게 보이는 유일한 이름이다.
_Avoid_: 기수, 과정, 코호트, cohort

**기수**:
LG전자 클래스에 붙는 회차 번호(1기, 2기, …). 클래스 이름의 일부일 뿐, 클래스 자체를 가리키는 말이 아니다. 인프런처럼 회차 없이 하나로 존재하는 클래스도 있다.

**운영자 클래스**:
운영자만 속하는 클래스(toycrane). 수강생이 새로 고를 수 있는 대상이 아니지만, 보드에서는 소속 프로젝트를 걸러 볼 수 있다.

**추천**:
프로젝트에 보내는 지지 표시. 프로젝트 순위를 정하는 기준이다.
_Avoid_: 투표, 좋아요, vote
